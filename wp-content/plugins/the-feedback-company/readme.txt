== Installation ==
1. Upload the `the-feedback-company` folder and all its contents to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the settings
4. Enter te necessary fields
5. ...Go baby go...
